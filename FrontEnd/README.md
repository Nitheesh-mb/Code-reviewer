# CodeReview Pro

A full-stack MERN application for performing automated code reviews and tracking analysis history.

## Features
- Secure User Authentication (JWT)
- Real-time Code Analysis
- Language Selection Support
- Persistent Review History
- Modern Responsive Dashboard

## Tech Stack
- **Frontend**: React, Tailwind CSS, Lucide Icons
- **Backend**: Node.js, Express, MongoDB
- **AI Integration**: Google Generative AI

## Installation

### Backend
1. Go to `BackEnd` folder
2. Run `npm install`
3. Create `.env` file with `GOOGLE_GEMINI_KEY`, `MONGO_URI`, and `JWT_SECRET`
4. Run `npm start`

### Frontend
1. Go to `FrontEnd` folder
2. Run `npm install`
3. Run `npm run dev`
